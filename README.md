# 🏠 Приложение для управления умным домом Stets Home

## Описание проекта
Производитель устройств для умного дома Stets планирует разработать собственное мобильное приложение.

## Цель 
Спроектировать требования для первой версии мобильного приложения Stets Home - от выявления потребностей заказчика и пользователей до передачи готовых артефактов команде разработки.

## Функциональные возможности (MVP)

**Учётная запись**
- Регистрация, вход, восстановление пароля

**Управление домом**
- Несколько домов на одного пользователя
- Совместный доступ к дому для членов семьи

**Устройства**
- Добавление умных лампочек по QR-коду или вручную
- Включение/выключение устройства
- Режим энергосбережения

**Комнаты**
- Создание комнат по типу с автоматическим названием
- Добавление устройств в комнату
- Управление всеми устройствами комнаты одной кнопкой

**Сценарии автоматизации**
- Создание сценариев с выбором устройств и команд
- Запуск вручную и по расписанию
